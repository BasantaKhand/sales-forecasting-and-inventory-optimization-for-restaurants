# Sales Forecasting & Inventory Optimization System

**Restaurant Predictive Analysis — Individual Thesis Project**

A full-stack system for restaurant sales forecasting and inventory optimization
using predictive analysis (ARIMA + Facebook Prophet), built for a Nepali
restaurant context.

## Tech Stack

- **Frontend:** React + Tailwind CSS (Vite), Recharts
- **Backend:** Python Flask (REST API)
- **Database:** MongoDB
- **ML Models:** ARIMA (statsmodels) + Facebook Prophet

## Dataset

`data/nepali_restaurant_sales_data.csv` — 206K+ sales records spanning 3 years
of Nepali restaurant transactions.
